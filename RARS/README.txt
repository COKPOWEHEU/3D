Демо-код расчета трехмерной графики на ассемблере RISC-V под эмулятор RARS.

rars_3D_pyramid.asm - простейший код вращающейся пирамиды
rard_3D_shadow.asm  - вращающаяся моделька мракориса
rars_3D_lur.asm     - вращающаяся моделька шныга с выводом времени итераций

Настройки симулятора: подключить графический дисплей размером 256х256 по адресу 0x10000000. Для rars_3D_lur можно сделать дисплей вдвое выше, тогда будет отображаться еще и задний буфер.
Для rars_3D_lur также надо подключить таймер, для остальных не обязательно.

Все расчеты ведутся в числах с фиксированной точкой, отсечение перекрывающихся граней осуществляется по методу z-buffer.
